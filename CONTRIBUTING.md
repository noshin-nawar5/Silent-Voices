# Contributing to Silent Voices 🤙

Thank you for your interest in contributing! Here's how to get started.

## 🌸 Ways to Contribute

- 🐛 Report bugs via [Issues](../../issues)
- 💡 Suggest features via [Issues](../../issues)
- 🔧 Submit pull requests
- 📖 Improve documentation

## 🚀 Development Setup

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Silent-Voices-.git
   cd Silent-Voices-
   ```
3. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. Make your changes
5. Commit using [Conventional Commits](https://www.conventionalcommits.org/):
   ```bash
   git commit -m "feat: add webcam support"
   git commit -m "fix: correct confidence rounding"
   git commit -m "docs: update API reference"
   ```
6. Push and open a Pull Request

## 📋 Commit Message Format

```
type: short description

Types: feat | fix | docs | style | refactor | test | chore
```

## ✅ Pull Request Checklist

- [ ] Code follows the existing style
- [ ] Self-reviewed the changes
- [ ] No breaking changes (or clearly documented)
- [ ] README updated if needed

## 🐛 Reporting Bugs

Open an issue and include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots if applicable
